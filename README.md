# Model Comparison

A comparison of how different AI models solve the same coding challenge: build a playable Snake game as a single HTML file.

## Structure

Each directory holds one model's attempt:

- `claude-sonnet/` — Claude Sonnet 5's attempt (`snake.html`)
- `qwen-3/` — Qwen 3.5 9B's attempt (`snake.html`)

Open any `snake.html` directly in a browser to play that model's version.
